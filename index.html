Bienvenue sur Manger Local

Grâce à l'application Manger Local, il n'a jamais été aussi simple de découvrir les producteurs présents autour de soi! Géolocalisez-vous en un tour de main, et commencez dès à présent à Manger Local!

Manger Local est une plateforme participative ayant pour but de regrouper les producteurs locaux ainsi que leurs points de vente.

Vous êtes producteur, vous connaissez un producteur, ou vous vendez des produits locaux? Un formulaire est à votre disposition afin de partager un nouveau point de vente.

Téléchargez dès maintenant l'application

A propos

Manger Local est une application communautaire permettant aux consommateurs de retrouver les producteurs proposant : la vente directe, la cueillette de leurs produits, les marchés, les magasins de producteurs…

Chacun est libre de poposer un lieu de vente par l'intermédiaire du formulaire 'Ajouter un lieu'. Cependant les modérateurs du site se réservent un droit de revue sur les propositions avant leur publication. Pour modifier un producteur déjà enregistré, veuillez utiliser le formulaire 'Modifier un lieu de vente '.

Manger Local est disponible uniquement sur Android, pour le moment!
